# 暮光星辰游戏端
TwilightStars_Client

你可以在[这里](https://github.com/Deemai/TwilightStars_Client/releases)下载到最新的暮光星辰整合包:yum:

使用开源的HMCL启动器构建Minecraft-1.16.5 Je游戏端整合包
  - 内置Fabric加载项
  - 内置Naitu Server更新源
  - 内置Naitu Skin Yggdrasil API
  - 内置TwilightStars服务器地址

整合包内已包含模组
  - 体素地图
  - 建筑辅助
  - 无缝加载
  - 界面微调
  - 血量显示
  - 边拿边走
  - 高清修复
